                        中英机器翻译项目说明
================================================================================

【项目概述】
本方案的核心特点：
1. 深层GRU架构（2层）- 探索RNN网络的效果
2. 指数Teacher Forcing衰减 - 平滑的训练策略过渡
3. 位置编码对比实验 - Sinusoidal vs Learned


================================================================================
                              使用方法
================================================================================

1. 训练深层GRU：
   python train.py gru

2. 训练Transformer（正弦PE）：
   python train.py trans

3. 运行PE对比实验：
   python train.py compare

4. 推理：
   python inference.py --model gru --ckpt ./ckpts/gru.pt
